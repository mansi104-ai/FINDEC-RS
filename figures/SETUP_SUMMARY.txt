## Troubleshooting

### Issue: "Package graphicx Error: File 'figures/system_architecture.png' not found"

**Solution:**
```
1. Check file exists: c:\Users\mansi\FINDEC-RS\figures\system_architecture.png
2. Ensure filename matches EXACTLY (case-sensitive)
3. Try re-running: pdflatex FINDEC-RS_Proposal.tex
```

### Issue: Figure looks blurry in PDF

**Solution:**
```
1. Export at 300 DPI (not 72 DPI)
2. Image should be at least 1600px wide
3. Re-export and recompile
```

### Issue: LaTeX won't find pdflatex

**Solution:** Ensure MikTeX or TeX Live is installed and in PATH
```bash
# Check if installed
pdflatex --version

# If not found, install MikTeX from: https://miktex.org/
```
